Limpeza de Dados do Dispositivo

Remova todo o conteúdo armazenado pelo AppBase neste dispositivo.

Atenção

Esta ação é irreversível. Todos os perfis, configurações, registros de telemetria e sessões autenticadas serão removidos. Após a limpeza você precisará configurar o aplicativo novamente.

Recomendamos exportar um backup antes de realizar a limpeza.
